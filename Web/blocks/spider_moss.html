Spider Moss

Spider Moss is a decorative block that generates naturally in areas where spiders have established their territory. It serves as both an environmental indicator of spider presence and a blue moss block to use in builds.

Obtaining

Natural Generation

Spider Moss generates naturally in caves underground. More specifically, it can be discovered in:

  • Surrounding Spider Eggs in Spider Egg Clusters
Breaking

Spider Moss can be harvested with any tool, all having the same efficiency. It drops itself when broken and can be collected for decorative purposes.

Usage

Decoration

Spider Moss serves primarily as a decorative building block, offering a unique dark aqua coloration that complements blue builds and underground environments.

Spreading

You can use bonemeal on spider moss to spread it. Unlike moss blocks, spider moss can spread downward into the ground.
